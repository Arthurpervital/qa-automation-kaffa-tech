# Instruções para rodar o projeto

# Requisitos mínimos para utilização do Cypress:

- Principais Requisitos e Pré-requisitos:
- Node.js: Ambiente necessário, de preferência em versões LTS (18+ recomendado).
- Gerenciador de Pacotes: npm (instalado com Node.js) ou yarn.
- Sistema Operacional:
- Windows: 10 ou 11 (64-bit).
- macOS: Versão 13.5 ou superior.
- Linux: Ubuntu 22.04+, Debian 11+, ou Fedora 42+.
- Editor de Código: Visual Studio Code (VS Code) é o mais recomendado.
- Hardware Mínimo: Processador x64 ou arm64. 

# Instação do Cypress

1. Abra um terminal e rode o seguinte comando:
 npm install cypress
 
 Esse comando irá criar a pasta **node_modules** 

2. Após, abra o cypress utilizando o comando
 npx cypress open
 Essa ação irá abrir o **cypress e criar a pasta do cypress no projeto**
